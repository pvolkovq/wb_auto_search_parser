# wb_auto_search_parser


Простой и быстрый парсер товаров WB по переданному запросу

**Запуск**

* Установить UV:

      curl -LsSf https://astral.sh/uv/install.sh | sh
  ссылка на документацию по установке https://docs.astral.sh/uv/getting-started/installation/

* В директории с проектом вызови команду ``uv venv`` она создаст окружение
* Подтяни связанные зависимости - ``uv sync``
* Запусти скрипт командой ``uv run main.py``
* Скрипт запустится и предложит ввести поисковый запрос. Введи Tenet t4 или то что тебе нужно
* Результаты выполнения скрипта будут лежать в ``result.xlsx``, рядом с ``main.py``
* Пример с собранными данными уже лежит в ``result.xlsx`` в репозитории
